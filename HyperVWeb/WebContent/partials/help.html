Configuration options

Address

You can choose Hyper-V Web Interface web server ip address and port in ListeningUrl option
ListeningUrl: use http://0.0.0.0:80/ or specify already assigned to this machine ip and port.
Hint: use ipconfig /all to get info about your networks.

Authentification

Auth: Possible values are: Off, Basic, Windows

Basic

If Basic auth enabled then usernames and password from BasicAuthUserCredentials are used.
Format: username:password

Windows

If Windows auth enabled then users from AllowedWindowsUsers are used. Format: computername\username
